# SURO VPN

Landing page prototype for a VPN service, built with React and Vite.

The project represents a clean, modern interface focused on clarity, trust, and conversion. It serves as a visual and structural foundation for a future VPN product.

Design by https://t.me/hencio

## Overview

SURO VPN is a front-end prototype of a marketing landing page.  
It demonstrates layout, visual style, and key user flows without backend integration.

The goal is to provide a ready-to-use UI base that can be extended into a full product.

## Features

- Animated hero section with primary value proposition  
- Server locations showcase  
- FAQ section with accordion behavior  
- Final call-to-action block  
- Integrated local assets (fonts, icons, flags, images)

## Tech Stack

- React  
- Vite  

## Notes

This is a UI prototype.  
No real VPN functionality, authentication, or infrastructure is included.

Further development should focus on:
- backend logic  
- connection handling  
- security and scaling
